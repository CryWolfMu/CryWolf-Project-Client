# CryWolf Project Installer 🐺

Ласкаво просимо! Це офіційний інсталятор нашого проекту.

### Як встановити:
1. Завантажте `CryWolf_Setup.exe` з розділу **Releases**.
2. Запустіть файл та слідуйте інструкціям.
3. Прийміть **Кодекс Поведінки** (це важливо для нашої спільноти).
4. Запустіть лаунчер з робочого столу.
