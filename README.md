### Як встановити:
1. Завантажте `CryWolf_Setup.exe` з розділу **Releases**.
2. Запустіть файл та слідуйте інструкціям.
3. Прийміть **Кодекс Поведінки** (це важливо для нашої спільноти).

Welcome to the official repository for the **CryWolf Project** game client. This repository hosts the latest stable versions of our installer.

## 📥 How to Install
1. Navigate to the **[Releases](https://github.com/CryWolfMu/CryWolf-Project-Client/releases/)** section on the right.
2. Download the `CryWolf_Setup.zip` (or `.exe`) file.
3. Run the installer and follow the on-screen instructions.
4. **Important:** You must read and accept our **Community Code of Conduct** during installation.
5. Launch the game using the **CryWolf Launcher** shortcut on your desktop.
