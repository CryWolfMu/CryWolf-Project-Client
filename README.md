## 📥 How to Install
1. Navigate to the **Releases** section on the right.
2. Download the `CryWolf_Setup.zip` (or `.exe`) file.
3. Run the installer and follow the on-screen instructions.
4. **Important:** Register your account on our [Website](https://crywolf.online/register) before logging in.
5. Launch the game using the **CryWolf Launcher** shortcut on your desktop.
